# 🎯 인사이드잡 (InsideJob)
## 현직자와 구직자를 연결하는 취업 멘토링 플랫폼

**현재 상태**: 🚀 **98% 완성** - 즉시 베타 서비스 런칭 가능  
**개발사**: (주)오픈커넥트  
**개발 기간**: 7일 (2025.08.24 ~ 2025.09.03)  
**기술 스택**: React 18, Tailwind CSS, Supabase

---

## 📋 **프로젝트 개요**

인사이드잡은 취업을 준비하는 구직자와 해당 업계의 현직자를 연결하여 **실무 중심의 커리어 멘토링**을 제공하는 플랫폼입니다.

### 🎯 **핵심 기능**
- **🧮 취업경쟁력 진단**: 3단계 100점 체계로 객관적 평가
- **💬 구직자 커뮤니티**: 5개 취업 관련 카테고리별 정보 공유
- **👨‍💼 현직자 매칭**: 10개 업계 현직자와 1:1 매칭  
- **💬 실시간 채팅**: 예약 후 전용 채팅방에서 상담
- **💳 투명한 수수료**: 20% 플랫폼 수수료 (관리자 조정 가능)
- **📊 관리자 시스템**: 완전한 플랫폼 운영 관리 도구

## 🎯 **주요 기능 상세**

### **취업경쟁력 진단 시스템**
```
1단계: 기본자격 평가 (30점)
├── 나이: 22-25세(5점) ~ 36세+(1점)  
├── 학력: 고졸(2점) ~ 대학원(10점)
├── 어학: TOEIC 등급별 점수 (10점)
└── 자격증: 컴활/정보처리기사 등 (5점)

2단계: 실무역량 평가 (40점)
├── 기술 숙련도 (15점): Python, Java, JS 등 최대 3개
└── 실무 경험 (25점): 인턴십, 프로젝트, 공모전 등

3단계: 시장적합성 + 준비도 (30점)  
├── 목표 직종 채용 수요 (10점)
├── 지원 전략 유연성 (10점)
└── 취업 준비도 (10점): 이력서/면접/네트워킹
```

**결과 제공**:
- **S~F 등급**: 100점 만점 6단계 평가
- **취업 성공률**: 10~95% 예측
- **예상 구직기간**: 2~12개월
- **맞춤 개선방안**: 점수 기반 추천사항

### **구직자 커뮤니티**
- **취업정보**: 채용공고, 업계 트렌드 공유
- **면접후기**: 실제 면접 경험담
- **자기계발**: 스킬업, 자격증 정보
- **네트워킹**: 동종업계 구직자 연결
- **고민상담**: 취업 관련 질의응답

### **현직자 매칭 시스템**
- **10개 업계**: IT, 금융, 마케팅, 의료, 교육, 제조, 서비스 등
- **유연한 상담시간**: 10분~60분 선택 가능
- **투명한 수수료**: 상담료의 20% (실시간 표시)
- **검증 시스템**: 관리자 승인을 통한 현직자 자격 확인

## 🏗️ **기술 아키텍처**

### **프론트엔드**
- **React 18**: CDN 방식으로 빠른 로딩
- **Tailwind CSS**: 완전한 반응형 디자인
- **JSX + React.createElement**: 혼재 방식 (안정적 동작)
- **단일 HTML 파일**: 875KB SPA 구조

### **백엔드**
- **Supabase**: PostgreSQL + Auth + Realtime
- **15개 테이블**: 완전한 비즈니스 로직 지원  
- **RLS 정책**: 사용자별 데이터 격리 보안
- **자동 트리거**: 수수료 계산 등 자동화

### **데이터베이스 구조**
```sql
-- 핵심 테이블들
user_profiles          -- 사용자 프로필
career_calculations    -- 진단 결과  
job_seeker_posts       -- 커뮤니티 게시글
professional_requests  -- 현직자 등록 요청
consultation_bookings  -- 상담 예약
chat_messages         -- 실시간 채팅
member_messages       -- 1:1 쪽지
system_settings       -- 시스템 설정
```

## 🔧 **개발 환경 설정**

### **필수 요구사항**
- **웹서버**: Python 3.7+ 또는 Node.js 14+
- **브라우저**: Chrome, Firefox, Safari, Edge (최신 버전)
- **Supabase 계정**: 데이터베이스 연동 시 필요

### **Supabase 설정 (선택사항)**
```bash
# 1. DATABASE_SCHEMA.sql 실행 (Supabase SQL Editor)
# 2. 환경변수 설정 확인
SUPABASE_URL=your_supabase_url
SUPABASE_ANON_KEY=your_anon_key
```

### **로컬 개발**
```bash  
# 개발 서버 실행
python3 -m http.server 8080 --bind 0.0.0.0

# 모바일 테스트 (같은 네트워크)
http://[PC-IP]:8080
```

## 📱 **사용자 가이드**

### **구직자 사용법**
1. **회원가입** → 이메일 인증
2. **취업경쟁력 진단** 완료 (필수)
3. **커뮤니티** 참여 (정보 공유)
4. **현직자 찾기** → 상담 예약
5. **채팅 상담** → 실시간 멘토링

### **현직자 사용법**
1. **현직자 등록** → 자격 증빙 제출
2. **관리자 승인** 대기
3. **상담 서비스** 등록 (시간/가격 설정)
4. **예약 접수** → 구직자 매칭
5. **채팅 상담** → 멘토링 제공

### **관리자 사용법**
- **현직자 승인**: 자격 검증 및 승인/거절
- **시스템 설정**: 수수료율, 상담시간 등 조정
- **사용자 관리**: 권한 변경, 통계 모니터링
- **플랫폼 운영**: 전체 서비스 현황 관리
